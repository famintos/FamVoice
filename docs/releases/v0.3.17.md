### Added

- Added microphone selection in Settings — choose which input device to use instead of always defaulting to the system default.
- Added 14 new transcription languages: Arabic, German, Spanish, French, Hindi, Italian, Japanese, Korean, Dutch, Polish, Russian, Turkish, Ukrainian, and Chinese.
- Added Groq `whisper-large-v3` as a model option alongside the existing turbo variant, for higher accuracy when speed is less critical.

### Changed

- The hint shown after a successful transcription now reads "Pasted to your app." when auto-paste is enabled, making it clear the paste already happened.

### Fixed

- Fixed the microphone stream not releasing between recordings — the mic icon no longer stays active in the system tray while the app is idle.
- Fixed trailing ellipsis (… or ...) sometimes appended by Groq when speech ends without continuation being passed through to the output.
- Fixed API error details being suppressed in release builds, making it easier to diagnose issues like quota limits or unavailable models.
