### Changed

- Improved Groq Whisper transcription hints for Portuguese dictation with a built-in vocabulary for common coding terms, tools, commands, and Faminto product names.
- Prioritized your saved Glossary entries in the transcription hint before the built-in vocabulary, so custom spellings such as project names and commands get stronger guidance.
- Removed the Transcription Hint toggle from Settings. FamVoice now applies the best available transcription hint automatically when the selected language supports it.

### Fixed

- Reduced common dictation mistakes around mixed Portuguese and developer terms such as FamVoice, FamSpace, Groq, npm commands, Cargo commands, clipboard, hotkey, and source file paths.
