### Added

- Added `whisper-large-v3` back as a Groq transcription option for higher-accuracy dictation when you want an alternative to `whisper-large-v3-turbo`.

### Changed

- Removed the unused `gpt-4o-mini-transcribe` and `gpt-4o-transcribe` transcription options so the model list stays focused on the speech-to-text paths that are still useful in FamVoice.
- Kept glossary replacements local to FamVoice instead of sending them as transcription prompt hints, while still applying your saved replacements to the final transcript.

### Fixed

- Reduced microphone startup latency by requesting a lower-latency input buffer, which helps capture the beginning of speech more reliably after pressing the hotkey.
- Restored microphone release between recordings so FamVoice stays out of the way when it is idle.
