## Agent Delegation

Subagents may be used whenever they materially improve parallelism, review quality, or execution speed. Explicit per-turn permission is not required.

## Subagent Reasoning Effort

Use `medium` by default for subagents.

Use `low` for quick repository exploration, narrow lookups, and simple localized tasks where fast turnaround matters more than deep synthesis.

Use `high` only for ambiguous debugging, architectural analysis, complex reviews, or high-risk changes where the cost of shallow reasoning is materially higher.
